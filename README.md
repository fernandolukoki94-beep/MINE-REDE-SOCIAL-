# Mini Rede Social (PWA)

Uma aplicação de rede social minimalista e funcional, construída com tecnologias web puras (HTML, CSS e JavaScript). Esta aplicação funciona inteiramente no navegador, utilizando o `localStorage` para persistência de dados, o que a torna ideal para demonstrações, protótipos ou uso pessoal offline.

## 🚀 Funcionalidades

- **Publicações com Imagem:** Partilha os teus pensamentos e inclui links de imagens para tornar os teus posts mais visuais.
- **Sistema de Likes e Comentários:** Interage com as publicações de forma simples.
- **Mensagens Rápidas:** Um chat simplificado para mensagens instantâneas.
- **Modo Escuro/Claro:** Suporte para temas visuais para melhor conforto em qualquer ambiente.
- **Filtro de Conteúdo:** Filtro automático de palavras impróprias para manter um ambiente saudável.
- **Pesquisa em Tempo Real:** Encontra rapidamente publicações ou autores específicos.
- **PWA (Progressive Web App):** Pode ser instalada no teu dispositivo e funciona offline graças ao Service Worker.

## 🛠️ Tecnologias Utilizadas

- **HTML5:** Estrutura semântica da aplicação.
- **CSS3:** Design moderno e responsivo com variáveis CSS para temas.
- **JavaScript (Vanilla):** Lógica de negócio, manipulação do DOM e gestão de estado local.
- **LocalStorage:** Armazenamento persistente no lado do cliente.
- **Service Workers:** Suporte para funcionamento offline e instalação como PWA.

## 📦 Como Utilizar

Não é necessária qualquer instalação de servidor ou dependências externas.

1. Clone o repositório:
   ```bash
   git clone https://github.com/fernandolukoki94-beep/MINE-REDE-SOCIAL-.git
   ```
2. Abra o ficheiro `index.html` em qualquer navegador moderno.
3. (Opcional) Para a funcionalidade de PWA, sirva os ficheiros através de um servidor local (ex: Live Server do VS Code).
